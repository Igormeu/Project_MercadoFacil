MercadoFacil - Projeto Android (pronto para abrir no Android Studio)

Como usar:
1. Baixe e extraia o ZIP do projeto.
2. Abra o Android Studio -> Open -> selecione a pasta 'MercadoFacil'.
3. O Android Studio pode pedir para atualizar o Gradle plugin; aceite as sugestões.
4. Build -> Run (aplicativo usa minSdk 24).
5. Login: preencha usuário e senha com qualquer texto para entrar (login fictício).
6. Para integrar backend: os pontos de entrada estão em LoginActivity (colocar autenticação real) e nos Adapters/Activities onde os dados são preparados.

Observações:
- MPAndroidChart já está na dependência; sincronize Gradle para baixar.
- Ícones são placeholders; substitua em res/mipmap e res/drawable pelos seus ativos.
